Tag < p >

Utilidade

  • A tag < p > é usada para definir um parágrafo de texto.
Exemplo: 
< p > Este é um parágrafo exemplo. < /p >
Subtags relacionadas:
  • Não possui subtags diretas, mas pode conter outras tags inline como < a >, < strong >, < em >, etc...
Resumo:
  • Usada para estruturar texto em blocos de parágrafos.

Tag < img >

Utilidade:
  • A tag < img > é usada para incorporar imagens em uma página web.
Exemplo: 
< img src="imagem.png" alt="Descrição da imagem" >
Atributos principais:
  • src: Especifica o caminho da imagem.
  • alt: Fornece uma descrição alternativa da imagem para acessibilidade e SEO.
  • width: Define as dimensões da Imagem.
Resumo:
  • Exibe imagens na página, importante para conteúdo visual e acessibilidade.

Tag < hr >

Utilidade:
  • A tag < hr > é usada para criar uma linha horizontal, geralmente usada para separar conteúdos ou seções.
Exemplo: 
            < p > Primeira seção < /p >
            < hr >
            < p > segunda seção < /p >
Atributos principais:
  • width, size, color: Pode ser estilizada usando CSS para ajustar a aparência.
Resumo:
  • Insere uma linha horizontal para separar visualmente o conteúdo.

Tag < h1 >

Utilidade:
  • A tag < h1 > é usada para definir o título mais importante de uma página ou seção.
Exemplo: 
 < h1 > Este é o título da página < /h1 >
Subtags relacionadas:
  • < h2 >, < h3 >, < h4 >, < h5 >, < h6 >: Outras tags de cabeçalho com hierarquia de importância decrescente.
Resumo:
  • Define o título prncipal, ajudando na estruturação e SEO. É o cabeçalho mais importante em uma hierarquia de headings.

Tag < br >

Utilidade:
  • A tag < br > é usada para inserir uma quebra de linha dentro da linha de um bloco de texto.
Exemplo: 
< p > Primeira linha < br >
      Segunda linha < /p >
Resumo:
  • Cria uma quebra de linha de texto, útil para formatar texto sem usar parágrafos separados.
