Tag < del >

Utilidade:
  • A tag < del > é usada para indicar que um texto foi deletado ou removido, geralmente exibido com uma linha cruzando o texto.
Exemplo: 
 < p > Este texto foi < del > removido < /del > alterado. < /p >
Resumo:
  • Marca o texto removido, útil para mostrar alterações ou correções em documentos.

Tag < small >

Utilidade:
  • A tag < small > é usada para reduzir o tamanho do texto em relação ao texto adjacente, geralmente para notas de rodapé ou informações secundárias.
Exemplo: 
< p > Este é um texto principal, e < small >este é um texto menor. < /small >< /p >
Resumo:
  • Exibe o texto em um tamanho menor, útil para destacar informações secundárias ou notas.

Tag < mark >

Utilidade:
  • A tag < mark > é usada para destacar ou marcar partes do texto, geralmente exibido com um fundo amarelo ou outra cor de destaque.
Exemplo: 
 < p > Este é um < mark > texto destacado < /mark > para chamar a atenção. < /p >
Resumo:
  • Destaca texto para chamar a atenção, útil para mostrar partes importantes ou relevantes em um documento.

Tag < strong >

Utilidade:
  • A tag < strong > é usada para dar ênfase ao texto, geralmente exibido em negrito. Também indica que o texto tem uma importância especial em termos de significado.
Exemplo: 
 < p > Este é um < strong > texto importante < /strong > que deve ser destacado. < /p >
Resumo:
  • Destaca o texto em negrito e indica importância, útil para enfatizar informações importantes.

Tag < em >

Utilidade:
  • A tag < em > é usada para enfatizar o texto, geralmente exibido em itálico. Indica uma ênfase semântica, sugerindo que o texto é importante ou tem um tom diferente.
Exemplo: 
< p > Este é um < em > texto enfatizado < /em > que merece atenção esecial < /p >
Resumo:
  • Destaca o texto em itálico e indica ênfase, útil para mostrar importância ou alterar o tom do texto.
